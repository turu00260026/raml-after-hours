# RAML 肩上ポートレート表情差分

## 共通仕様

- 用途: ゲーム上段の会話演出ステージ
- 構図: 肩から上、頭頂と両肩を入れる、頭上の余白は最小
- 表情: `neutral` / `concern` / `analysis` / `alert`
- 配置: RAML側または会話の主役を左、相手役または敵を右
- スチール画像が指定されたイベントはスチールを優先
- 顔、髪型、衣装、年齢、装備、画風は元立ち絵を維持

## 生成方法

組み込みの画像生成を使用し、元立ち絵をID・衣装・画風の参照画像として指定した。各キャラにつき2x2表情アトラスを生成し、次の順序で分割する。

1. 左上: `neutral`
2. 右上: `concern`
3. 左下: `analysis`
4. 右下: `alert`

背景はゲーム画面になじむ濃い青緑。文字、枠、ラベル、透かしは入れない。

## アルパカ

ショウの牧場イベント用に、同じ肩上規格で白いアルパカの4状態を追加。通常は完全な無表情、`analysis` は任務説明を真顔で見定める表情にする。ショウを左、アルパカを右へ配置する。
